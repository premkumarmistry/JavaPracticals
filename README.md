## Getting Started

1. **Clone the repository**:
   ```sh
   git clone https://github.com/premkumarmistry/JavaPracticals.git
   cd JavaPracticals
   ```

2. **Explore the exercises**:
   - Navigate to the `src` directory to find exercises organized by topics.
   - Choose an exercise of interest and open the corresponding Java source file.

3. **Compile and run**:
   - Compile the Java source file using `javac` command.
   - Run the compiled class file using `java` command to see the output.
